# Desert Fury 🌵🎮

Desert Fury is a 3D endless runner game made with Unity.  
The player controls a rolling ball, avoids obstacles, collects coins, and survives as long as possible in a stylized desert environment.

## Features
- Endless runner gameplay
- Coin collection system
- Obstacles and score system
- Desert-themed environment
- Trees and cactus spawning
- Simple UI and Game Over screen

## Built With
- Unity 6
- C#

## Gameplay
The goal is to survive as long as possible while staying on the road and collecting coins.
